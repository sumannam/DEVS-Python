# Introduction 

DEVS-Python은 Python 기반으로 DEVS 모델들을 모델링하고 시뮬레이션한다. Python을 사용하는 이 시스템에서는 시스템의 구성을 기본 클래스 계층 구조에 따라 정의한다. 또한, Python의 장점을 그대로 이어받아 문법들이 단순하여 초보자도 쉽게 모델링이 가능하며 풍부한 라이브러리를 사용할 수 있다
